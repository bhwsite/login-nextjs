## üíª Getting started

### Requirements

- [Node.js](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/) or [npm](https://www.npmjs.com/)

**Access the folder**

```bash
$ cd login-nextjs
```

**Follow the steps below**

**Running application**

```bash
# Install the dependencies
$ yarn 

# Run the web app
$ yarn dev
```
